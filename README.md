# React Intensive

### 🛠️ Technology Stack

- TypeScript
- React
- Vite

### ⚙️ Installation

```bash
git clone https://github.com/expant/react-intensive.git
cd react-intensive
npm install
```

### 💻 Development

```bash
npm run dev
```

### 🏗️ Production Build

```bash
npm run build
npm run preview
```
